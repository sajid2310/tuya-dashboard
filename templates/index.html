Tuya Cloud API credentials

From the Tuya IoT Platform, create a Cloud project and copy its Access ID/Client ID and Access Secret/Client Secret below. Link your Tuya (Smart Life) app account to the project under "Devices → Link Tuya App Account" so your devices show up. This lets Sync pull device names and local keys automatically instead of you finding them by hand. Credentials are encrypted at rest in this app's local data/config.json file.
